## ðŸ§© Course Structure and Topics

The course is divided into multiple parts, each focusing on a specific area of full-stack development.

| Part | Topic Title | Core Concepts Covered |
|------|-------------|------------------------|
| **Part 0** | Fundamentals of Web Apps | Basic web application operations, HTTP GET/POST, Forms, SPA architecture |
| **Part 1** | Introduction to React | Component-based architecture, state, props, event handlers, debugging |
| **Part 2** | Communicating with the Server | Fetching data, rendering collections, axios/fetch, useEffect |
| **Part 3** | Programming Servers with Node.js and Express | Backend setup, routing, MongoDB |
| **Part 4** | Testing Express Apps & User Administration | Backend testing, API best practices, JWT authentication |
| **Part 5** | Testing React Apps | Unit tests, integration tests, E2E testing with Cypress |
| **Part 6** | Advanced State Management | Redux or Context API with useReducer |
| **Part 7** | React Router, Custom Hooks, Styling & Webpack | Client-side routing, reusable hooks, styling approaches |
| **Part 8** | GraphQL | Queries, mutations, building a GraphQL server |
| **Part 9** | TypeScript | Migrating JS projects to TS in both frontend & backend |
| **Part 10** | React Native | Cross-platform mobile development |
| **Part 11** | CI/CD | GitHub Actions and CI/CD setup |
| **Part 12** | Container Technology | Docker and container-based development |
| **Part 13** | Using a Relational Database | PostgreSQL, SQL, ORM (e.g., Sequelize) |
